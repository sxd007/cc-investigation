# /interview

调查访谈全流程支持 — 从策划到复盘。

## Usage
```
/interview plan             访谈策划（目的、问题、策略）
/interview guide            生成问话提纲
/interview analyze [text]   陈述分析（SCAN）
/interview transcript       笔录模板/格式化
/interview review [wp#]     审查已完成的访谈记录
```

## Process
遵循 PEACE 访谈模型，根据对象类型（证人/被调查人/举报人）调整策略。分析输出符合 interview-analysis 技能标准。
