# /report

调查写作与报告 — 底稿编制到正式报告全流程。

## Usage
```
/report memo [case#]        撰写中期备忘录
/report draft [case#]       起草调查报告
/report review [doc#]       审查/复核调查文书（写作质量检查）
/report visualize [data]    数据可视化（证据/时间线/资金流）
/report check [doc#]        写作规范检查清单
```

## Process
遵循 writing-reporting 技能的 SCQA 结构和金字塔原理。输出前自动运行写作风格检查清单。
