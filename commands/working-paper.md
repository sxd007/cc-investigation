# /working-paper

调查底稿规范化管理。

## Usage
```
/working-paper create [type] 创建新底稿（管理类/证据类/分析类/程序类/结论类）
/working-paper index [case#] 底稿索引
/working-paper review [wp#]  底稿复核（ALCOA 检查）
/working-paper export [case#] 导出完整底稿包
```

## Process
遵循 evidence-management 技能的 ALCOA 原则和底稿复核清单。
