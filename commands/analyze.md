# /analyze

数据分析工具 — 发现数据中的异常和模式。

## Usage
```
/analyze benford [file]     Benford定律首位数字分析
/analyze trends [file]      数据趋势分析
/analyze anomalies [file]   异常值检测
/analyze flows [data]       资金流向/流程分析
/analyze network [data]     关系图谱分析
```

## Process
根据 data-analysis 技能框架执行分析，输出可导入报告的结果和可视化建议。
