# 调查底稿标准

## 底稿编制标准 (ALCOA)
- **A**ttributable (可追溯): 每条记录注明编制人和日期
- **L**egible (可读): 排版清晰、语言准确
- **C**ontemporaneous (同步): 工作完成后立即编制
- **O**riginal (原始): 索引编号唯一、副本标注清晰
- **A**ccurate (准确): 数据准确、修改留痕

## 底稿内容要求
1. 每份底稿必须有唯一索引编号 (WP-YYYY-NNN)
2. 底稿必须包含: 案件编号、主题、编制人/日期、复核人/日期
3. 所有数据必须注明来源
4. 推理过程必须完整记录, 无逻辑跳跃
5. 负面/不支持的信息也必须记录

## 底稿分类
- 管理类: 立案、计划、进度、结案
- 证据类: 证据清单、保管链、鉴定
- 分析类: 数据分析、推理、流程图
- 程序类: 访谈记录、外勤记录
- 结论类: 发现汇总、结论依据

## 复核要求
1. 每份底稿必须有独立复核人
2. 复核需在底稿编制后7个工作日内完成
3. 复核发现的问题需跟踪整改
4. 复核人不得是底稿编制人的下属
