## A. 责任销售（sales_rep）

### A1 — 项目申报与终端客户确认

1. 该项目最初是谁提出或交给你跟进的？请按时间顺序说明从接触机会到项目申报的完整过程。
2. 终端客户是谁、由谁确认、你第一次确认其真实需求是在什么时间、通过什么方式？
3. 你是否亲自拜访、电话沟通或视频沟通过终端客户？请说明具体对象、时间、地点和沟通内容。
4. 你手上有哪些能证明终端客户真实存在并有采购意愿的材料？如邮件、会议纪要、名片、需求文档、报价确认等。
5. 如果终端客户信息来自代理商，请说明代理商是谁、如何提供、你做了哪些独立核验。
6. 若你无法说明终端客户确认过程，请解释原因；哪些环节是你亲自做的，哪些是你转述得知的？

**追问要点**：
- 回答若停留在“业务惯例”“代理商说的”“一直这么做”，必须追问到具体时间、对象、载体。
- 如其声称见过客户但无任何留痕，记录为证明力不足，并追问是否存在同行见证人或后续材料。

### A2 — 折扣申请依据

1. 你申请该项目折扣时提交了哪些依据材料？分别由谁准备、谁审核、谁最终提交？
2. 折扣申请中的价格压力、竞争对手信息、项目紧迫性是如何形成判断的？
3. 这些依据中哪些是你亲自核实过的，哪些是代理商、主管或他人转告的？
4. 若存在竞争情报，请说明信息来源、取得时间、原始载体和核验方式。
5. 折扣申请过程中是否有人建议你采用某种表述、删减某些事实或突出某些因素？
6. 回看当时申请，你认为其中哪些内容今天仍然准确，哪些可能存在不完整或偏差？

**追问要点**：
- 若其只陈述结论，不陈述形成过程，追问“你当时根据什么作出这个判断”。
- 若其推责为“模板这么写”“代理商提供的”，追问其本人审核责任和具体核验动作。

### A3 — 与代理商的沟通情况

1. 你与代理商在该项目中的分工是什么？谁负责客户沟通、谁负责报价、谁负责材料准备？
2. 你是否指导过代理商准备项目材料、合同描述、客户信息或成本说明？请举例说明。
3. 关于货物去向、签收安排、终端部署地点，你在什么时间知道了哪些信息？
4. 你与代理商就该项目有无微信、邮件、电话或线下沟通记录？哪些内容最关键？
5. 代理商是否曾向你表达过“这样写更容易过审”“不要写太细”“先这样报”等建议？
6. 如你声称对实际流向不知情，请解释在你的岗位职责下为何没有核验或跟进。

**追问要点**：
- 若其承认沟通存在但拿不出记录，追问记录保管方式、删除原因、是否存在对方侧证。
- 若其只承认外围协调，不回应是否指导造假，必须回到具体文件和具体表述追问。

### A4 — 项目成本构成

1. 项目中的 OtherCost 或第三方成本包含哪些具体项目？每项费用由谁提出、如何测算？
2. 第三方服务商是谁，由谁引入，与项目实际需求的关系是什么？
3. 这些服务是否实际发生过？对应的合同、交付件、验收记录、付款依据分别是什么？
4. 你是否见过或审核过第三方交付成果？若没有，你依据什么相信成本真实存在？
5. 若第三方与代理商、员工存在特殊关系，请说明你是否知晓、何时知晓、是否上报过。
6. 如现在无法说明费用形成过程，请指出你当时依赖的判断依据和责任分工。

**追问要点**：
- 对“服务真实发生”必须追问交付内容、交付时间、交付对象、验收方式。
- 对“金额合理”必须追问测算过程，而不是接受抽象评价。

### A5 — 异常情况的知情与处理

1. 你是否在项目推进过程中发现过客户信息、配置、交付、成本或渠道链路方面的异常？
2. 第一次发现异常是什么时候，由谁告诉你，或者你是通过什么材料意识到的？
3. 发现异常后你采取了哪些动作：核实、上报、纠正、继续推进，还是暂时搁置？
4. 是否有人明确或暗示你不要继续追问、不要记录、先把项目做完再说？
5. 如你未上报，请解释原因；是认为不构成问题、缺乏证据，还是担心其他后果？
6. 今天回看，哪些异常你认为是误会，哪些异常你认为本应进一步核查？

**追问要点**：
- 如其承认小问题但回避核心问题，记录为选择性配合，并继续围绕核心假设追问。

## B. 代理商责任人（agent_principal）

### B1 — 货物流向确认

1. 该项目货物从下单到最终交付的完整路径是什么？请按时间顺序说明每一站的主体、地点和交接方式。
2. 最终实际收货方是谁？与合同约定的终端客户是否一致？若不一致，请解释原因。
3. 你能提供哪些物流、仓储、签收、安装或交付凭证来证明货物流向？
4. 货物是否曾短暂停留在代理商仓库、其他中间商仓库或第三方地点？
5. 谁决定了交付路径和签收安排？销售是否参与或知情？
6. 若部分货物流向你无法说明，请解释是记录缺失、业务惯例，还是实际并非你方控制。

**追问要点**：
- 若只说“按正常流程交付”，必须追问单次具体交付的时间、地点、签收人和凭证。

### B2 — 终端客户的实际情况

1. 合同中的终端客户是谁？你方第一次接触该客户是在何时、通过谁、以何种方式？
2. 终端客户是否真实提出过采购、安装或使用需求？请说明具体场景。
3. 你能提供哪些材料证明终端客户真实参与了该交易，如合同、对账、验收、部署、售后记录？
4. 如果终端客户仅出现在合同或申报材料中，但实际接触有限，请说明原因。
5. 是否存在由其他渠道商、集成商或最终使用人实际承接货物或需求的情况？
6. 如你认为终端客户信息不便提供，请说明限制依据，并说明是否可提供脱敏或替代证明。

**追问要点**：
- 对“客户真实存在”必须追问其采购动作，而不是只确认主体名称存在。

### B3 — 发票开具对象

1. 该项目的发票分别开给了谁？与合同相对方、实际付款方、实际收货方是否一致？
2. 发票内容、金额、时间是如何确定的？是否有人要求按特定方式拆分或改写？
3. 若发票开具对象与合同约定不一致，请说明业务原因和审批依据。
4. 你能提供哪些发票、红冲、补开、付款或对账材料来解释这一路径？
5. 该项目是否存在由第三方代收代付、转开发票或先开后补合同的情形？

**追问要点**：
- 若其解释为“客户要求”或“财务安排”，追问具体提出人、审批链和留痕材料。

### B4 — 渠道链路

1. 除你方外，该项目是否还有其他中间商、服务商、挂靠主体或代操作主体参与？
2. 每个主体在该项目中扮演什么角色，拿走了哪一部分利润或费用？
3. 资金从终端客户到你方、再到其他主体的流向是什么？是否存在与合同链路不一致的支付安排？
4. 是否存在为满足申报、结算或交付需要而临时引入某主体的情形？
5. 哪些链路是销售知情或参与安排的，哪些是你方自行处理的？
6. 若你否认存在其他层级，请解释为何现有物流、发票、安装或付款线索显示出多层结构。

**追问要点**：
- 一旦发现“业务方便”“行业惯例”等概括性解释，必须追问具体主体和具体动作。

### B5 — 与销售的沟通指示

1. 销售在该项目中给过你哪些明确指示？涉及客户名称、合同内容、配置、成本或交付安排吗？
2. 是否有人建议你在合同、报价、说明材料中采用特定写法、隐藏某些信息或突出某些说法？
3. 这些指示是口头给出的，还是通过微信、邮件、电话、会议给出的？是否有留痕？
4. 当你对某些安排提出疑问时，销售或其主管是如何解释的？
5. 若你声称完全自主决定，请解释销售为何仍掌握关键项目信息或审批节奏。

**追问要点**：
- 对“没有明确指示”要继续追问是否存在默认共识、暗示性要求或惯用模板。

## C. 销售主管（sales_supervisor）

### C1 — 项目审批过程

1. 该项目折扣、客户、配置或成本审批经过了哪些环节？你本人在哪些节点参与决策？
2. 你审批时重点核验了哪些内容？哪些是你亲自看过的，哪些是依赖下属或代理商陈述的？
3. 对终端客户真实性、项目必要性、成本合理性，你采取过哪些实质性核验动作？
4. 审批过程中是否出现过异常提醒、补件要求或争议点？最后如何处理？
5. 若今天看存在问题，你认为当时的审批失效点在哪里：制度设计、执行疏漏，还是信息被误导？

**追问要点**：
- 若主管回答停留在“按流程审批”，必须追问其个人具体审查动作和判断依据。

### C2 — 异常信号的知情情况

1. 在项目推进过程中，你是否收到过任何来自销售、代理商、客户、财务、售后或渠道团队的异常反馈？
2. 这些异常第一次出现是什么时候，由谁提出，内容是什么？
3. 你当时如何判断这些异常的性质：普通业务问题、资料不完整，还是潜在违规？
4. 你采取了哪些后续动作？是否要求补核、暂停、复盘、上报或留痕？
5. 是否存在你知道异常但认为“先做成项目再说”或“金额不大不必扩大”的情况？

**追问要点**：
- 对“记不清”“没印象”要结合既有记录追问其为何对关键节点毫无记忆。

### C3 — 内控执行情况

1. 你平时如何管理下属在客户真实性、折扣申请、代理商合作和成本申报方面的风险？
2. 你团队是否有固定的高风险代理商、高风险项目类型或常见操作偏差？
3. 对下属与代理商的私人往来、利益冲突、异常合作方式，你了解多少、如何管控？
4. 你是否要求过团队在材料表述、项目包装或审批应对上采取特定策略？
5. 如果本案属实，你认为更像个体违规、团队惯性，还是内控缺失导致的系统性问题？

**追问要点**：
- 对“制度都有、都培训过”要追问实际执行、抽查、纠偏和追责记录。

## 通用问题（适用于所有角色）

1. 你刚才的哪些陈述是亲身经历，哪些是他人转述，哪些是你的判断或推测？
2. 对于你无法确认的部分，请明确说明是不知道、记不清、没参与，还是暂时拿不出材料。
3. 你是否还有任何能够支持你说法的补充材料、联系人或系统记录？
4. 如果今天需要你把整件事从头到尾讲给一位完全不了解背景的第三人，你认为最关键的三个事实是什么？
5. 你是否知悉公司关于真实性申报、廉洁从业、配合调查、材料完整性的基本要求？
6. 你是否愿意在访谈后补充提交材料或书面说明？预计何时可以提交？

**通用审查提示**：
- 对所有角色，都应优先要求其描述“过程”而不是只给“结论”。
- 对所有拒绝、模糊、回避和无法确认，都应记录原因，并评估是否构成对抗性行为或证明力不足。
