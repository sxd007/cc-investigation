# 费用报销舞弊调查

适用于企业内部费用报销环节的舞弊调查。核心矛盾：**报销申报的事由/金额与实际发生的费用之间的差异**。

费用报销舞弊属于 ACFE 资产挪用（Asset Misappropriation）— 开票方案（Billing Schemes）— 费用报销舞弊（Expense Reimbursement Frauds）类别。此类舞弊往往利用制度漏洞、审批惯性或单据审查不严实施。

## When to Activate

- 接到费用报销相关的举报或投诉（差旅费、招待费、办公采购等）
- 费用报销数据出现异常模式（高频、高额、集中于特定报销人）
- 内部审计发现报销凭证异常（连号发票、金额整数、缺少附件）
- 员工消费行为与职级/岗位职责不匹配
- 需要设计费用报销舞弊调查方案
- 跨部门数据比对发现报销数据与考勤/行程/业务记录不匹配

## 四种舞弊模式

### 1. 费用性质篡改 (Mischaracterized Expenses)

**模式描述**：员工将个人消费（私人购物、家庭旅行、个人餐饮等）申报为业务费用，利用审批人对事由的真实性难以逐一核实的特点套取公司资金。

**关键信号：**
- 节假日或非工作时间产生的报销（周末、年假期间）
- 报销事由与职级岗位职责明显不匹配（如行政岗产生大量客户招待费）
- 发票开具地与员工行程/考勤记录不匹配
- 票据品名为笼统类别（"办公用品""食品""日用品"）而无明细清单
- 电子发票占比过高且无对应的消费凭证

**调查切入点：**
- 交叉比对报销时间与考勤记录、出差申请单、日历日程
- 核验发票开具地的合理性（是否在出差/会议/客户所在地）
- 要求提供大额"办公用品""食品"类报销的消费明细清单
- 比对同岗位、同级别员工的人均报销水平

### 2. 费用金额虚增 (Overstated Expenses)

**模式描述**：员工在真实发生的业务费用基础上虚报金额，通过伪造/篡改收据、虚构消费明细或与商家合谋抬价后分成的方式牟利。

**关键信号：**
- 票据金额为整数或接近审批阈值的金额（如 4999、995）
- 同一商家同期为多人出具金额接近的票据
- 费用金额明显超出同类消费的市场合理水平
- 票据存在涂改、拼接、二次复印的痕迹
- 同一商家出具的票据编号不连续但日期接近

**调查切入点：**
- 核验大额票据的真实性（联系商家验证、比对发票系统记录）
- 比对同一线路/酒店的多人报销金额，识别异常偏高者
- 检查纸质票据的物理特征（纸质、印刷质量、字体一致性）
- 对比报销金额与银行扣款记录或信用卡账单

### 3. 虚构费用 (Fictitious Expenses)

**模式描述**：员工凭空捏造从未发生的费用，使用虚假票据或虚构的租赁/服务合同套取公司资金。常涉及与外部商户合谋或使用假发票。

**关键信号：**
- 发票在税务系统查询结果为"查无此票"或状态异常
- 商家工商信息存疑（新注册、无实际经营地址、已被吊销）
- 报销事项无对应的业务活动记录（无出差记录却报销差旅费）
- 合同/订单对应的服务或货物无法验收确认
- 连续、周期性的固定金额报销（警惕定期定额的"习惯性"报销）

**调查切入点：**
- 通过税务局网站或发票查验平台核验发票真伪
- 核验供应商工商登记信息和实际经营状态
- 要求提供费用发生的佐证材料（签到表、会议议程、照片、邮件往来）
- 追踪报销资金去向（是否与供应商再回流给员工）
- 比对合同中留存的联系方式（电话/邮箱）是否与员工个人关联

### 4. 重复报销 (Duplicate Reimbursement)

**模式描述**：员工将同一笔费用通过不同途径或在不同时间多次申报，利用信息不对称和财务审核周期差重复获取报销款。

**关键信号：**
- 同一金额/日期/商家的组合在报销记录中多次出现
- 同一笔费用同时出现在差旅报销和项目费用报销中
- 提交电子版和纸质版各一次（"双轨提交"）
- 利用不同报销类型申报同一费用（"业务招待费"+"培训费"）
- 多笔报销在时间、金额、用途上高度相似

**调查切入点：**
- 对比所有报销记录中的发票号码，识别重复使用的发票
- 按报销人+金额+日期+商家四维组合去重比对
- 检查是否同一笔费用由不同人同时申报（如多人 AA 制聚餐每人全额报）
- 按时间线追踪同一事项是否被多次报销

## 调查方案设计要点

### 信号交叉验证

费用报销舞弊的四种模式可能共存，也可能与其他舞弊类型关联：

| 初始信号 | 通常伴随的模式 | 可能关联的舞弊类型 |
|---------|-------------|-----------------|
| 发票真伪存疑 | 虚构费用 + 可能涉及供应商合谋 | 利益冲突（如供应商为员工关联方） |
| 金额接近审批阈值 | 金额虚增 + 性质篡改 | — |
| 节假日报销 | 性质篡改 | — |
| 同一发票号多次出现 | 重复报销 | — |
| 高频小额报销 | 虚构费用 / 性质篡改 | — |

### 数据分析切入点

费用报销舞弊调查中，数据分析是最高效的筛查手段：

| 分析维度 | 方法 | 预期发现 |
|---------|------|---------|
| **报销人维度** | 按人聚合金额、频次、日均消费，识别异常离群 | 费用模式异常的个体 |
| **时间维度** | 按周/月/季分析报销分布，识别规律性报销模式 | 虚构费用的周期特征 |
| **商家维度** | 按供应商聚合，识别异常的集中采购或关联交易 | 合谋供应商或虚假商家 |
| **审批链维度** | 分析审批通过率、审批速度，识别异常宽松的审批节点 | 串通审批或审批失效环节 |
| **跨部门比对** | 报销记录 vs 考勤/行程/门禁/加班记录 | 费用发生时员工不在场的异常 |
| **Benford 分析** | 对报销金额进行首位数分布检验 | 人工捏造金额的非自然分布 |

### 数据来源分类

| 数据类别 | 内容 | 获取方式 |
|---------|------|---------|
| **系统内数据** | 报销系统记录、考勤记录、差旅申请、门禁记录、OA 审批流 | 系统查询或导出（可在静默期完成） |
| **接触获取数据** | 发票原件、消费明细清单、合同/订单、验收记录 | 需接触财务/报销人，在 FIELDWORK 获取 |
| **公开查询数据** | 发票真伪查验、供应商工商信息、市场价格比对 | 公开渠道查询（可在静默期完成） |

### 调查原则

调查用到的系统内数据遵循 **双轨获取** 原则：

- **主路径（primary）**：通过系统接口或 SQL 脚本自动获取，结果可重复验证
- **备用路径（fallback）**：主路径不可用时，由调查员从系统手动提取，结果须人工核验
- **不可获取时**：标注数据缺口（evidence_gap），评估对 finding confidence 的影响

详细的通用调查原则（静默边界、缺口归因、信源前缀）参见 [`skills/investigation-foundation/SKILL.md`](../investigation-foundation/SKILL.md)。

## 分析辅助工具

以下工具类型可辅助加速费用报销数据分析。**它们不是必须的**——未配置时，由模型按上述方法直接完成分析。

### 数据库查询类 MCP（如配置）

- **能力：** 执行 SQL 查询、导出报销数据
- **辅助场景：** 批量提取报销明细、跨系统（报销/考勤/门禁）关联分析
- **不可用时：** 从报销系统手动导出数据，用电子表格或文本编辑器分析

### 网络搜索类 MCP（如配置）

- **能力：** 公开信息检索
- **辅助场景：** 发票真伪查验、市场价格比对、供应商背景调查
- **不可用时：** 浏览器手动查询税务局网站和工商信息

### 推理验证类 MCP（如配置）

- **能力：** 分步逻辑推演、多路径比对
- **辅助场景：** 报销行为模式分析、资金回流路径推演
- **不可用时：** 由模型自行完成推演，输出推演过程供审查
