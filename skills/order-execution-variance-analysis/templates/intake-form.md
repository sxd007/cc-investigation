# 项目执行差异分析 — 调查员信息收集清单

当调查员需要开展项目执行差异分析时，agent 应主动向调查员收集以下信息。
可以直接把本清单发给调查员填写，或逐项询问。
本表同时适用于单项目分析（填一个项目）和多项目整合分析（按项目逐一填写）。
**调查员提供的信息越完整，agent 修改次数越少。**

## 填写提示

1. 如果某条链路在某一层没有节点（如申报链路没有三代），该格留空即可，图中会自动显示占位符和"未申报"/"未涉及"标签
2. 交易数据（成交金额、利润、利润率）是交易的边的属性标签
3. 节点的附加信息是节点自己的属性，分开展示，展示在节点框下方
4. 角色说明尽量简短（≤8字），如"行业金牌"、"非认证"、"申报终端"、"实际使用方"
5. 风险点的"具体描述"建议包含对比数字，如"申报 2.9% → 实际 4.3%"，比"利润率不符"更有说服力
6. **维度标签**区分合同流、资金流、货物流、控制关系——图中会用不同颜色和线型区分，便于一目了然地看出哪种关系出现了偏差
